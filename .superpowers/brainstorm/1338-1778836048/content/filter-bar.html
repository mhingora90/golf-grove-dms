Filter Bar Design

All filtering is server-side (fast at 100s of leads). Which set of filters does your team need?

A

Essential — Stage + Search

Search box + stage filter tabs (All / New / Contacted / …). Simple, fast, covers 90% of daily use.

| All (142) New (38) Contacted (24) No Response (31) Site Visit (12) Follow-Up (19) Won (9) Lost (9)
B

Standard — Stage + Source + Assigned

Search + stage dropdown + source dropdown + assigned-to dropdown. Lets sales managers filter by team member. Most useful for teams.

Stage: All ▾ Source: All ▾ Assigned: All ▾ | + Add Lead

Stage options: All / New / Contacted / No Response / Site Visit / Follow-Up / Won / Lost
Source options: All / Meta Ads / Website / Referral / Walk-In / Other

C

Full — Stage + Source + Assigned + Date range

Everything in B plus a date range picker (created_at). Good for reporting — "show all Meta leads this month".

Stage: All ▾ Source: All ▾ Assigned: All ▾ May 1 → May 15 📅 | + Add Lead
